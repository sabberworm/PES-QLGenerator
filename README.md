# PES-QLGenerator

Use PESHandler to Generate QuickLook Previews of Brother PES Embroidery Files.

## Usage

1. Install `PES-QLGenerator.qlgenerator` into `~/Library/QuickLook`.
2. Preview a `.pes` file.
